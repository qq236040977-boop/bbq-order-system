📋 首次设置步骤

  1. 1 安装 Termux
    从 F-Droid 或官网下载安装 Termux
  2. 2 传项目到手机
    把整个 bbq-order-system 文件夹传到手机
  3. 3 安装 Node.js
    在 Termux 中运行：
    pkg install nodejs
  4. 4 启动服务器
    进入项目目录：
    cd bbq-order-system
    npm install
    node server.js
  5. 5 安装 ngrok
    在 Termux 中：
    pkg install ngrok
    然后在 ngrok.com 注册免费账号获取 token
    ngrok config add-authtoken 你的token
  6. 6 启动 ngrok 隧道
    新开一个 Termux 会话：
    ngrok http 3000
  7. 7 生成二维码
    把 ngrok 给的地址（如 https://xxx.ngrok-free.app）填到上面，生成二维码，打印贴桌上
💡 每天营业第一步：
打开 Termux → 启动服务器 → 启动 ngrok → 刷新本页面确认二维码 → 开张！
